# Todo App Frontend

A simple Next.js frontend for the Todo API.

## Prerequisites

- Node.js 18+
- FastAPI backend running at `http://localhost:8000`

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production Build

```bash
npm run build
npm start
```

## Features

- View all todos
- Add new todos
- Toggle completion status
- Delete todos
- Filter by status (All / Active / Completed)
